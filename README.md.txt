# TinyURL - Full Stack WebApp

## Overview
A TinyURL clone with:
- Angular frontend
- ASP.NET Core Minimal API backend
- Azure SQL + Blob + Function + Web App
- Terraform IaC
- GitHub Actions CI/CD

## Features
- Shorten URLs (public/private)
- List, search, delete, copy
- Click tracking
- Auto-delete all URLs every hour
- Logging to Azure Blob

## Prerequisites
- Node.js 18+
- .NET 8 SDK
- Azure CLI
- Terraform
- GitHub Account
